This plugin allow you to insert in your posts or pages a Gantt chart, based on the gantti class wrote by bastianallgeier. Fully localized

== Description ==

You can create as many gantt as you like, then you insert a shortcode in post or page and the gantt we'll be rendered.
Based on the [gantti class](https://github.com/bastianallgeier/gantti "Gantti class") by [Bastian Allgeier](http://bastianallgeier.com  "Bastian Allgeier")
It has been tested only on WP 3.4 and up, maybe works with older versions too.


== Installation ==

1. Upload `WPIT GANTT` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create at least a GANNT and insert the right shortcode in a page or post.
